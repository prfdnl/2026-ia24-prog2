# Atividade 2 - Laboratório de Programação

## Objetivo

- Treinar a capacidade de interpretação de código
- Treinar a capacidade de implementação de funcionalidades
- Treinar a capacidade de ler e adaptar código de terceiros
- Treinar a capacidade de usar o `git` e o `github` para controle de versão e entrega de atividades

## Descrição

Na pasta `tri2.ativ2` existe o projeto parcial que fizemos em sala de aula, onde estamos adicionando a capacidade de nosso sistema salvar informações em um  banco de dados `sqlite`, o objetivo é ler esse código, entender seu funcionamento e reescrever o métodos do `ToDo List` para usar o banco de dados ao invés de um arquivo `json`.
